## 📖 אודות הפרויקט

**עץ חיים** היא מערכת ניהול לימוד מתקדמת המיועדת למעקב אחר התקדמות בלימוד הש"ס הבבלי וששת סדרי המשנה. המערכת מספקת כלים חזקים למעקב אישי, ניהול משתמשים, ודוחות התקדמות מפורטים.

---

## ✨ תכונות עיקריות

### 🎯 למשתמשים
- **מעקב התקדמות אישי** - סימון דפי גמרא ומשניות שנלמדו
- **שמירת מיקום אחרון** - המשך מהמקום האחרון שבו עצרת, מכל מכשיר
- **סטטיסטיקות מפורטות** - מעקב אחר אחוזי התקדמות בכל מסכת וסדר
- **ממשק אינטואיטיבי** - עיצוב נקי וידידותי למשתמש בעברית
- **גרירה מהירה** - סימון מרובה של דפים בגרירת עכבר

### 🕯️ מערכת הקדשות
- **הקדשות דינמיות** - לעילוי נשמת ולרפואת חולים
- **עיצוב מותאם אישית** - בחירת פונט, צבע, גודל ויישור
- **ניהול זמנים** - הגדרת תאריך תפוגה אוטומטית להקדשות
- **תצוגה חזותית** - עיצוב מרהיב עם אייקונים ואנימציות

### 👨‍💼 לאדמינים
- **ניהול משתמשים** - יצירה ועריכה של משתמשים ותפקידים
- **דוחות התקדמות** - צפייה בנתונים של כל התלמידים
- **גרפים ויזואליים** - מעקב חזותי אחר אחוזי סיום
- **ממשק ניהול הקדשות** - הוספה, עריכה ומחיקה של הקדשות

### 🔒 אבטחה
- **אימות reCAPTCHA v3** - הגנה מפני בוטים ומתקפות
- **ניהול הרשאות** - הפרדה בין תלמידים, מנהלי קבוצות ואדמינים
- **שמירת נתונים בענן** - גיבוי אוטומטי ב-PythonAnywhere

---

## 🛠️ טכנולוגיות

### Frontend
- **React 18** - ספריית JavaScript מודרנית לבניית ממשק משתמש
- **Tailwind CSS** - framework CSS לעיצוב מהיר ומותאם
- **Babel Standalone** - טרנספייליציה של JSX בדפדפן

### Backend
- **Flask** (Python) - framework web קל משקל
- **SQLAlchemy** - ORM לניהול מסד נתונים
- **SQLite** - מסד נתונים מוטמע
- **Flask-CORS** - תמיכה ב-Cross-Origin Resource Sharing

### Security & Hosting
- **Google reCAPTCHA v3** - אימות אנטי-בוט
- **PythonAnywhere** - אירוח Backend
- **GitHub Pages** - אירוח Frontend (אופציונלי)
